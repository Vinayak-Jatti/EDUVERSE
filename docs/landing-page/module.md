# Landing Page Module Overview

## 1. Description
The digital doorstep of EDUVERSE. A high-performance, dark-themed responsive destination designed for complex desktop-first interactions and mobile-first readability.

## 2. Business Rules
- **Aesthetic**: Deep Black (`#000000`) with a White/Indigo palette.
- **Background**: Mandatory Graph Style Grid Pattern.
- **Logo**: 🎓 Graduation Cap symbol integrated into the core brand system.
- **Tagline**: "Made by students, made for students."

## 3. Key Components
- **Hero**: Cursor-responsive parallax background. Motion tracking on X/Y axis with spring damping.
- **Features**: Rotating border hover effects (`conic-gradient`) on feature nodes.
- **Apply Form**: Resume-first career ingress point.

## 4. Logic Flow
1. User enters Landing Node.
2. Background grid & interactive particles initialize.
3. Feature nodes update with "Exact Fair Features" (Squads, TechNews).
4. Career form allows signal broadcast (apply/submit).
